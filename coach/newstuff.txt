Creating a React Native app for tracking game time and bench time for players on a team is a great idea! I'll break this down into key steps and provide you with the necessary code for setting up the app.

### App Features

1. **Player List**: You should have a list of players that the coach can add or remove.
2. **Track Time**: Each player’s game time and bench time should be tracked in real-time.
3. **Timer**: A simple timer to start and stop game time for each player.
4. **Data Storage**: Use local storage or a simple database to store the time data.

### Steps

1. **Create a New React Native App**

   First, let's create a new React Native project.

   ```bash
   npx react-native init GameTimeTracker
   cd GameTimeTracker
   ```

2. **Install Dependencies**

   You'll need a few dependencies like `react-navigation` for screen navigation and `react-native-async-storage` for saving data locally.

   ```bash
   npm install @react-navigation/native @react-navigation/stack react-native-async-storage/async-storage
   npx pod-install ios
   ```

3. **Create Screens**

   You'll need two main screens: one for viewing and managing the players and another for tracking time.

   #### File: `App.js`

   Start with a basic structure.

   ```javascript
   import React, { useState, useEffect } from 'react';
   import { View, Text, Button, TextInput, FlatList, TouchableOpacity } from 'react-native';
   import AsyncStorage from '@react-native-async-storage/async-storage';

   export default function App() {
     const [players, setPlayers] = useState([]);
     const [playerName, setPlayerName] = useState('');

     useEffect(() => {
       const loadPlayers = async () => {
         const savedPlayers = await AsyncStorage.getItem('players');
         if (savedPlayers) {
           setPlayers(JSON.parse(savedPlayers));
         }
       };
       loadPlayers();
     }, []);

     useEffect(() => {
       const savePlayers =d async () => {
         await AsyncStorage.setItem('players', JSON.stringify(players));
       };
       savePlayers();
     }, [players]);

     const addPlayer = () => {
       if (playerName.trim()) {
         const newPlayer = { id: Date.now().toString(), name: playerName, gameTime: 0, benchTime: 0 };
         setPlayers([...players, newPlayer]);
         setPlayerName('');
       }
     };

     return (
       <View style={{ flex: 1, padding: 20 }}>
         <Text style={{ fontSize: 20, fontWeight: 'bold' }}>Player List</Text>
         <TextInput
           style={{ borderWidth: 1, marginBottom: 10, padding: 8 }}
           placeholder="Enter Player Name"
           value={playerName}
           onChangeText={setPlayerName}
         />
         <Button title="Add Player" onPress={addPlayer} />
         <FlatList
           data={players}
           keyExtractor={(item) => item.id}
           renderItem={({ item }) => (
             <TouchableOpacity
               style={{ padding: 10, borderBottomWidth: 1 }}
               onPress={() => navigation.navigate('PlayerDetail', { player: item })}
             >
               <Text>{item.name}</Text>
             </TouchableOpacity>
           )}
         />
       </View>
     );
   }
   ```

   ### Explanation:

   * **`players`**: State for storing player data, including game and bench time.
   * **`playerName`**: State for capturing the new player's name.
   * **`useEffect`**: Loads and saves player data from/to `AsyncStorage`.

4. **Create Player Detail Screen for Tracking Time**

   Now, create a separate screen where the coach can track game and bench time.

   #### File: `PlayerDetail.js`

   ```javascript
   import React, { useState, useEffect } from 'react';
   import { View, Text, Button, TouchableOpacity } from 'react-native';
   import { useNavigation, useRoute } from '@react-navigation/native';

   const PlayerDetail = () => {
     const { params } = useRoute();
     const { player } = params;
     const [gameTime, setGameTime] = useState(player.gameTime);
     const [benchTime, setBenchTime] = useState(player.benchTime);
     const [timer, setTimer] = useState(null);
     const [isPlaying, setIsPlaying] = useState(false);
     const navigation = useNavigation();

     useEffect(() => {
       if (isPlaying) {
         const interval = setInterval(() => {
           setGameTime((prevTime) => prevTime + 1);
         }, 1000);
         setTimer(interval);
       } else {
         clearInterval(timer);
       }

       return () => clearInterval(timer);
     }, [isPlaying]);

     const startStopTimer = () => {
       setIsPlaying((prev) => !prev);
     };

     const handleBench = () => {
       if (isPlaying) {
         setIsPlaying(false);
         setBenchTime((prev) => prev + 1);
       } else {
         setGameTime((prev) => prev + 1);
       }
     };

     const handleSave = () => {
       const updatedPlayer = { ...player, gameTime, benchTime };
       navigation.goBack();
     };

     return (
       <View style={{ flex: 1, padding: 20 }}>
         <Text style={{ fontSize: 24, fontWeight: 'bold' }}>{player.name}</Text>
         <Text>Game Time: {gameTime} seconds</Text>
         <Text>Bench Time: {benchTime} seconds</Text>
         <Button title={isPlaying ? 'Stop Timer' : 'Start Timer'} onPress={startStopTimer} />
         <TouchableOpacity onPress={handleBench} style={{ marginTop: 20 }}>
           <Button title="Switch to Bench" />
         </TouchableOpacity>
         <Button title="Save" onPress={handleSave} style={{ marginTop: 20 }} />
       </View>
     );
   };

   export default PlayerDetail;
   ```

   ### Explanation:

   * **`gameTime` and `benchTime`**: Track the player’s time during the game and while on the bench.
   * **`startStopTimer`**: Starts and stops the timer.
   * **`handleBench`**: Switches the player to the bench and starts tracking their bench time.
   * **`handleSave`**: Saves the player's updated data and navigates back.

5. **Set Up Navigation**

   You need to set up navigation between the screens.

   #### File: `AppNavigator.js`

   ```javascript
   import React from 'react';
   import { createStackNavigator } from '@react-navigation/stack';
   import { NavigationContainer } from '@react-navigation/native';
   import App from './App';
   import PlayerDetail from './PlayerDetail';

   const Stack = createStackNavigator();

   const AppNavigator = () => {
     return (
       <NavigationContainer>
         <Stack.Navigator initialRouteName="Home">
           <Stack.Screen name="Home" component={App} />
           <Stack.Screen name="PlayerDetail" component={PlayerDetail} />
         </Stack.Navigator>
       </NavigationContainer>
     );
   };

   export default AppNavigator;
   ```

6. **Main Entry Point**

   Now, modify your `index.js` to use the `AppNavigator`.

   #### File: `index.js`

   ```javascript
   import { AppRegistry } from 'react-native';
   import AppNavigator from './AppNavigator';
   import { name as appName } from './app.json';

   AppRegistry.registerComponent(appName, () => AppNavigator);
   ```
